github page+hexo构建静态博客

2020-12-26
搭建

首先，完成整套的搭建需要：

1.下载安装git工具下载地址并添加环境变量
2.下载安装node.js下载地址
3.github上有个自己的账号

具备以上条件，我们就可以开始搭建自己的静态博客了

打开git bash命令行：

1.更换npm镜像为淘宝镜像：
npm config set registry http://registry.npm.taobao.org/
2.下载hexo：
npm install -g hexo
3.初始化hexo：
自己新建个文件夹，示例中我新建了myblog：

进入文件夹，打开gitbash，初始化hexo:hexo init，成功后myblog文件夹多了些东西：

4.对_config.yml进行基础配置：

（这里新版本的theme移到文件最后去了，如果如图所示配置的话，需要将后面重复的theme删除）
这里主题theme我选择了concise，更多主题移步：hexo主题选择（其中有些主题对hexo有别的配置要求，详见每个主题的github）
下载我选择的主题:
git clone https://github.com/sanonz/hexo-theme-concise.git themes/concise
下载后theme文件夹有：

5.本地查看：
运行hexo g && hexo s，构建并本地启动，默认端口4000，访问localhost:4000
6.git pages部署：

1.新建仓库：


前后名称要保持一致，然后点击创建仓库按钮
2.修改_config.yml：

repo写你刚刚创建的仓库地址
3.安装hexo自动部署、发布工具（在my page文件夹下运行）：
npm install hexo-deployer-git --save
4.发布到git：
hexo clean && hexo g && hexo d 清除public，构建，deploy到git
5.访问git page：
https://你的用户名.github.io/
